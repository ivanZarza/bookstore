BOOKSTORE

Aplicación de coleccion de libros, desarrollada con Angular, React y Vue.js. Cada una de las aplicaciones tiene su propio diseño y funcionalidad, pero todas comparten la misma base de datos y API.

Angular

Aplicación de colección de libros desarrollada con Angular, un framework robusto y escalable. Esta aplicación utiliza TypeScript como lenguaje principal, lo que permite un desarrollo más estructurado y seguro. Además, se implementaron componentes reutilizables y un sistema de enrutamiento dinámico para una navegación fluida. La aplicación está diseñada con un enfoque en la reactividad y el rendimiento, aprovechando las capacidades de RxJS para manejar flujos de datos asíncronos. También se integraron servicios para consumir la API compartida, garantizando una comunicación eficiente con la base de datos. El diseño sigue las mejores prácticas de Angular, asegurando una experiencia de usuario moderna y funcional.

React

Aplicación de colección de libros desarrollada con React y TypeScript, utilizando JSX para la creación de componentes dinámicos y reutilizables. La validación de formularios se implementó de manera versátil, combinando el uso de React Hook Form, Zod para validaciones basadas en esquemas, y validaciones personalizadas hechas a mano. El diseño de la interfaz se realizó con Tailwind CSS, lo que permitió un desarrollo rápido y un diseño moderno y responsivo. La aplicación sigue un enfoque modular, asegurando una experiencia de usuario fluida y un código mantenible.

Vue

Aplicación de colección de libros desarrollada con Vue, aprovechando mis conocimientos previos en este framework. Para el diseño de la interfaz, se utilizó Vuetify, un poderoso sistema de componentes que permitió crear una experiencia de usuario moderna, responsiva y consistente. La aplicación sigue un enfoque basado en componentes reutilizables, lo que facilita la escalabilidad y el mantenimiento del código. Además, se implementaron funcionalidades dinámicas y reactivas, características clave de Vue, para garantizar una navegación fluida y una interacción intuitiva con el usuario.
