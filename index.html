BOOKSTORE

Esta es una aplicación de colección de libros que desarrollé para practicar y mejorar mis habilidades en Angular, React y Vue.js. Cada una de las aplicaciones tiene su propio diseño y funcionalidad, pero todas comparten la misma base de datos y API. El backend está desplegado en Vercel, y las diferentes páginas de frontend están configuradas con GitHub Actions para despliegues automáticos. Este proyecto me permitió explorar conceptos avanzados de desarrollo frontend y backend.

Angular

Desarrollé esta aplicación con Angular, un framework robusto y escalable. Utilicé TypeScript como lenguaje principal, lo que me permitió escribir un código más estructurado y seguro. Implementé componentes reutilizables y un sistema de enrutamiento dinámico para una navegación fluida. También aproveché las capacidades de RxJS para manejar flujos de datos asíncronos y aseguré una comunicación eficiente con la API compartida. Este proyecto me ayudó a profundizar en las mejores prácticas de Angular y a crear una experiencia de usuario moderna y funcional.

React

Esta aplicación fue desarrollada con React y TypeScript, utilizando JSX para crear componentes dinámicos y reutilizables. Practiqué la validación de formularios combinando React Hook Form, Zod para validaciones basadas en esquemas, y validaciones personalizadas hechas a mano. Además, utilicé Tailwind CSS para diseñar una interfaz moderna y responsiva. Este proyecto me permitió explorar un enfoque modular y asegurar una experiencia de usuario fluida y un código mantenible.

Vue

En esta aplicación utilicé Vue, aprovechando mis conocimientos previos en este framework. Para el diseño de la interfaz, implementé Vuetify, un sistema de componentes que me permitió crear una experiencia de usuario moderna y responsiva. Este proyecto me ayudó a practicar el enfoque basado en componentes reutilizables y a implementar funcionalidades dinámicas y reactivas, características clave de Vue. Además, exploré cómo garantizar una navegación fluida y una interacción intuitiva con el usuario.
